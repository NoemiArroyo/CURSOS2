Curso: Programción Web

Descripción del Curso

El curso de programación web es una capacitación diseñada para enseñar a los estudiantes a desarrollar aplicaciones y sitios web utilizando una variedad de tecnologías y lenguajes de programación. Estos cursos suelen abarcar conceptos y habilidades fundamentales necesarios para diseñar, crear y mantener sitios web dinámicos y funcionales.

Fecha de Inicio y Finalización

Inicio: 10/11/2023

Finalización: 10/02/2024

Objetivos del Curso

  • Crear páginas web con HTML.
  • Aplicar estilos y diseño con CSS.
  • Programar interactividad con JavaScript.
  • Desplegar aplicaciones web y gestionar hosting.

Temas del Curso

  • Fundamentos de desarrollo web
  • HTML (HyperText Markup Language)
  • CSS (Cascading Style Sheets)
  • JavaScrip
  • Frameworks y bibliotecas
  • Backend y servidores
